# NickelProject

NickelProject是基于leanote的.net core发行版

## 介绍

使用.net Core搭建~~漫画网站~~笔记托管服务，



**特性**

* 兼容：与Leanote笔记兼容
* 管理：更强大的托管后台
* 保密：允许使用密码加密笔记或者限制访问


#### 软件架构
前端框架： ~~AmazeUI +JQuery1.11.1~~ LeanoteUI

后端框架： ~~Asp.net Core 2.2  (standard2.0)~~ .net Core 3.1

服务器端： ~~Window Server+IIS~~ ubuntu18 lST  

数据库端： ~~MySQL~~ PostgreSQL

缓存服务： Redis  

分布式节点：暂无

#### 安装教程

```ssh
git clone URL
cd  NickelProject
dotnet run
```
